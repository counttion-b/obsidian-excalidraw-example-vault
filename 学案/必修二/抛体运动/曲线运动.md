## (1)曲线运动的特点与条件
### 1.曲线运动的运动学特征
曲线运动中速度的方向是时刻改变的，质点在某一点 （或某一时刻）的瞬时速度的方向在曲线的这一点的切线上.(生活经验和数学推理)

曲线运动一定是变速运动.
### 2.曲线运动的动力学特征
1.曲线运动的物体合外力$F_合$一定不为0.

![[Pasted image 20251217105443.png]]

2.本质上来说,$F_{\parallel}$改变速度大小,$F_{\perp}$改变速度方向.

3.$F_合$与速度$v$不在同一条直线上.$F_{\perp} \neq 0$.

4.$F_合$指向轨迹曲线凹向.

## (2)曲线运动的处理方法-运动的合成与分解
### 物理思想
研究比较复杂的运动时，常常把这个运动看作是两个或几个比较简单的运动组成的，使问题变得容易研究.这与力的正交分解类似,都是**将复杂的问题分解成多个简单问题的组合**.这种化简的想法是重要的物理思维.

**二维运动的分析方法:运动的合成与分解**

运动的合成与分解指的其实就是**速度,位移**的合成与分解.
### 例题
> [!ti] *例题*
> 一人一猴在玩杂技，如图所示，直杆长为 $L$ ，猴子由静止开始，以加速度 $a$ 沿直杆匀加速向上爬，同时人顶着直杆水平匀速运动．已知在一段时间内，猴子由 A 运动到 B，而人也由甲位置运动到了乙位置。请根据以上信息完成下列任务：
（1）建立恰当的坐标系，在图中定性画出猴子运动的轨迹
（2）分别计算时间为$t$时,猴子水平方向和竖直方向的位移.
（3）计算猴子运动的轨迹方程 $y=f(x)$.
>    （4）求轨迹上任意一点猴子速度的大小和方向.
>    ![[Pasted image 20251217225434.png|250]]
### 方法描述
在这个实例中，我们看到猴子向右上方的运动可以看成由沿杆向上的运动和水平向右的运动共同构成。猴子沿杆向上的运动和它随着杆向右的运动，都叫作分运动；而猴子相对于地面向右上方的运动叫作合运动。

由分运动求合运动的过程，叫作运动的**合成**；

由合运动求分运动的过程，叫作运动的**分解**。

运动的合成与分解遵从矢量运算法则。
### 性质
（1）等效性：合运动是由各分运动共同产生的总运动效果，各分运动的总的运动效果与合运动可以相互替代．

（2）独立性：各分运动独立进行，不受其他分运动的影响．

（3）等时性：合运动和各分运动经历的时间相等.联系两个分运动

## (3)小船过河
前提条件:$v_船$大小恒定,方向任选,$v_水$大小恒定,方向沿河岸(故不会影响垂直河岸方向的运动).

$$
\text { 小船过河: }\left\{\begin{array}{l}
1 . \text { 求最短渡河时间 } t_{\min }: \text { 船头垂直于河岸; } t_{\min }=\frac{d}{v_{\text {船 }}} \\
\text { 2. 求最短渡河路程 } S_{\min }:\left\{\begin{array}{l}
(1) v_{\text {船 }}>v_{\text {水 }}: \text { 合速度垂直于河岸 } ; S_{\min }=d \\
(2) v_{\text {船 }}<v_{\text {水 }}: \text { 船速垂直于合速度 }(\text { 切线 }) ; S_{\min }=\frac{v_{\text {水 }}}{v_{\text {船 }}}
\end{array}\right.
\end{array}\right.
$$

## (4)速度关联问题
人以恒定的速度$v_0$拉绳时,求此时小物块的速度$v$.
![[Pasted image 20251220002059.png]]

0.(基础知识)沿绳方向速度相等

1.物体运动的合速度为：物体对地的**真实**运动方向

2.沿绳方向$v_{\parallel}$:伸缩绳,使绳长变化.

垂直绳方向$v_{\perp}$:旋转绳,使角度变化.

3.由速度的矢量三角形可得:$v_{\parallel}=v_0$,$v_合=\frac{v_0}{cos \theta}$

提醒:运动的合成与分解即**速度**的合成与分解,与力的合成与分解不同.
