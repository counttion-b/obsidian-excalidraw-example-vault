### (1)连线上场强变化规律
连线上场强的变化规律图像如图（b）所示,定义水平向右为正方向.
![[Pasted image 20251126212356.png]] 连线上场强的变化规律:
$M O N$ 场强的变化：从 $M \rightarrow O \rightarrow N$ ，场强先减小至 0 后反向增大， 0 点场强为 $0 ; M \rightarrow-\infty 、 N \rightarrow +\infty$ 的场强均逐渐减小．
### (2)垂直平分线上场强变化规律
垂直平分线上场强的变化规律图像如图（c）所示,定义竖直向上为正方向.
![[Pasted image 20251126220601.png]]
垂直平分线上场强的变化规律:
$A O B$ 场强的变化：从 $A \rightarrow O \rightarrow B$ ，场强先增大后减小至 0 再反向增大后再减小，$O$ 点场强为 0 ；当 $\angle A M O$ 的正弦值为 $\frac{\sqrt{3}}{3}$ 时，合场强大小最大．
### (3)对称点上的规律总结
由于电场线关于 $A O B 、 M O N$ 对称，即具有对称性，$a 、 b 、 c 、 d$ 为中心在 0 点的矩形的四个顶点，如图（d）所示．
![[Pasted image 20251126220903.png]]
①$a 、 b 、 c 、 d$ 的场强大小均相等；
② $a$ 与 $d 、 b$ 与 $c$ 的场强方向关于连线 MON 对称；
③ $a$ 与 $b 、 c$ 与 $d$ 的场强方向关于垂直平分线 $A O B$ 对称；
④ $a$ 与 $c 、 b$ 与 $d$ 的场强方向关于 0 点中心对称．

### （1）连线上场强的变化规律图像如图（b）所示,定义水平向右为正方向.
![[Pasted image 20251126221216.png]] 连线上场强的变化规律:
$M O N$ 场强的变化：从 $M \rightarrow O \rightarrow N$ ，场强先减小后增大，$M O N$ 上 $O$ 点的场强最小，$A O B$ 上 $O$ 点的场强最大；$M$ 点左侧、 $N$ 点右侧的场强大小均逐渐减小。
### （2）垂直平分线上场强的变化规律图像如图（c）所示,定义水平向右为正方向.
![[Pasted image 20251126222023.png]]
垂直平分线上场强的变化规律:
$A O B$ 场强的变化：从 $A \rightarrow O \rightarrow B$ ，场强先增大后减小。
### （3）对称点上的规律总结
由于电场线关于 $M O N$ 对称，即具有对称性，$a$ 、 $b 、 c 、 d$ 为中心在 0 点的矩形的四个顶点，$a b$ 边与连线平行，如图（d）所示．
①$a 、 b 、 c 、 d$ 的场强大小均相等；
②$a$ 与 $c 、 b$ 与 $d$ 的场强方向相同；
③$a$ 与 $d 、 b$ 与 $c$ 的场强方向关于连线 MON 对称；
④关于 0 点对称的点的场强大小相等、方向相同．
