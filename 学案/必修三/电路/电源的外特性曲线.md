# 电源的外特性曲线 #h0
# 1. 知识讲解
## (1)公式推导与意义
由闭合电路欧姆定律推出,因为$I$与$U_外$这两个物理量相对好测量,故选其为横纵坐标,根据图像的斜率和截距,可以计算电源的电动势和内阻.
$$
U_{\text {外 }}=-r_{\text {内 }} I+E
$$

## (3)工作点
![[Pasted image 20260103013214.png|200]]
电源的电动势和内阻是不变的，正是由于外电阻的变化才会引起外电压 $U$ 和电流的变化．如果将两个图线放在一个 $U-I$ 图像中，两个图线的交点叫作＂工作点＂。工作点的物理意义是电阻和电源连接在一起时的路端电压和干路电流值 ．如下图所示．
![[Pasted image 20260103013231.png|200]]

### 工作点的能量计算
![[Pasted image 20260103013332.png|250]]
由工作点向 $U$ 轴和 $I$ 轴引垂线，和坐标轴围成的面积 ：

$$
S_1=U_0 \times I_0=P_{\text {出 }} \text {, 即输出功率; }
$$


过 $U$ 轴截距垂线，另过工作点作垂线 $I$ 轴垂线，两垂线和坐标轴围成的面积：

$$
\begin{gathered}
S_2=E \times I_0=P_{\text {总 }} \text {, 即总功率. } \\
S_2-S_1=P_{\text {总 }}-P_{\text {出 }}=P_{\text {内 }} \text {, 即内阻消耗功率. }
\end{gathered}
$$
