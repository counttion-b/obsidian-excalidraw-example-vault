## 1. 振动与简谐振动
### 1.1. 1.振动的基本概念
#### 1.1.1. 生活中的振动
#### 1.1.2. 机械振动的认识
1．机械振动 ：物体（或物体的某一部分）在某一中心位置两侧所做的往复运动，叫做机械振动，简称振动。

2．平衡位置 ：中心位置（振动停止时物体所在的位置）称为平衡位置。

### 1.2. 简谐振动
#### 1.2.1. 模型建构
（1）弹簧振子模型：如图所示，把一个有孔的小球装在**轻质弹簧**的一端，弹簧的另一端固定，小球穿在光滑的杆上，能够自由滑动，两者之间的**摩擦可以忽略**，弹簧的质量与小球相比也可以忽略。把小球拉向右方，然后放开，它就左右运动起来．小球原来静止时的位置叫做平衡位置，小球在平衡位置附近的往复运动，是一种机械振动，简称振动．小球和弹簧所组成的系统称为弹簧振子。

#### 1.2.2. 描述简谐振动的物理量
描述弹簧振子振动的物理量：
1、一般运动描述：位置、位移、时刻、时间、速度、加速度(描述状态)

2、守恒性：机械能守恒$\to$振幅 $\boldsymbol{A}$ (描述强弱))

3、周期性：周期 $T$ ，频率 $f$(描述快慢)

### 1.3. 2.运动学描述

1.振动位移

从平衡位置指向振子所在位置的有向线段.总是背离平衡位置

从1到2,振动位移是什么?

2.振幅A

（1）定义：振动物体离开平衡位置的最大距离。单位： m

（2）物理意义：描述振动强弱（能量）的物理量

（3）振幅与位移的区别：

振幅是$\_\_\_\_$(标量/矢量),振幅$\_\_\_\_$(会/不会)随小球的运动而变化.

位移是$\_\_\_\_$(标量/矢量),振幅$\_\_\_\_$(会/不会)随小球的运动而变化.

3.周期：完成一次全振动所用的时间

全振动：如果从振动物体向右通过 $O$ 的时刻开始计时，它将运动到 $M$ ，然后向左回到 $O$ ，又继续向左运动到达 $M^{\prime}$ ，之后又向右回到 $O$ 。这样一个完整的振动过程称为一次全振动


一次全振动所经过的路程是$\_\_\_\_$个振幅.

> [!question] 从$P_0$点向右出发做一次全振动,对应哪几段过程?

全振动：$x,v$ 都相同（大小、方向）的两点间的振动.

4.频率：$f=\frac{1}{T}$ ，表示单位时间内完成全振动的次数 单位：赫兹（hertz），符号是 Hz 。


### 1.4. 3.弹簧振子的位移-时间图像

#### 1.4.2. 简谐振动第一定义(运动学定义)：

如果物体的位移与时间的关系遵从正(余)弦函数的规律，即它的振动图像（ $x-t$ 图像）是一条正弦曲线，这样的振动是一种简谐运动 （simple harmonic motion）

做简谐运动的物体的位移 $x$ 与运动时间 $t$ 之间满足正弦函数关系，因此，位移 $x$ 的一般函数表达式可写为：$x=A \sin (\omega t+\varphi)$.
其中A是振幅,$\omega$是角频率,$\omega = \frac{2 \pi}{T}$,$\varphi$是初相位.
> [!quote] 
**相位和相位差**
频率相同的简谐振动才有比较相位的必要.
同相:振动步调一致
反相:振动步调刚好相反

## 2. 2.动力学和能量

### 2.1. 动力学特征
![[Pasted image 20260205160618.png|200]]
做简谐振动的物体为什么可以一直回到平衡位置?
1.简谐振动的回复力:使物体回到平衡位置的**效果力**.即振动方向上的合力,总指向平衡位置

> [!question] 我们还学过哪个效果力?
向心力:使物体做圆周运动的效果力,即半径方向上的合力,总指向圆心

回复力总与振动位移反向.
在弹簧振子中,弹簧弹力提供回复力$F_回=F_弹=k \cdot \Delta x=k \cdot (-x)$
即$F_回=-kx$.
2.简谐振动第二定义(动力学):
如果物体在运动方向上①所受的力与它偏离平衡位置位移的大小成正比，并且②总是指向平衡位置，即物体所受的合力具有 $F=-k x$的形式，则物体的运动就是简谐运动。
> [!ti] 例题
> 试证明:竖直方向的弹簧振子是简谐运动.

### 2.2. 简谐振动的能量特征
![[Pasted image 20260105234615.png|300]]

某量为最大值、最小值用 ＂最大＂和＂最小＂表示，某量为零用数字＂ 0 ＂表示，增加和减少分别用＂$\uparrow$＂和 ＂$\downarrow$＂ 表示（假设 $P 、 Q$ 为最大位移处）。

| 位置    | $Q$ | $Q \rightarrow O$ | $O$ | $O \rightarrow P$ | $P$ |
| :---- | :-- | :---------------- | :-- | :---------------- | :-- |
| 位移的大小 |     |                   |     |                   |     |
| 速度的大小 |     |                   |     |                   |     |
| 动能    |     |                   |     |                   |     |
| 弹性势能  |     |                   |     |                   |     |
| 机械能   |     |                   |     |                   |     |
结论：
（1）在弹簧振子运动的任意位置，系统的动能与势能之和都是一定的，遵守机械能守恒定律。
（2）当小球运动到最大位移时，动能为 0 ，弹性势能最大，系统的机械能等于最大弹性势能。
（3）对于弹簧劲度系数和小球质量都一定的系统，振幅越大，机械能越大。$E_机 \propto A^2$.
