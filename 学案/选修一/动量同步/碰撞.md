## 1. 碰撞规律
### 1.1. 碰撞
碰撞在物理学中表现为两粒子或物体间极短的相互作用。碰撞前后参与物发生速度，动量或能量改变。

这个过程中,
$$
\begin{aligned}
& I_{F_1}>0 \Rightarrow \text { 向A注入动量 } \\
& I_{F_2}<0 \Rightarrow \text { 从 } B \text { 取出动量 } \\
& I_{F_1}=-I_{F_2}\left(\sum I_{\text {内 }}=0\right)
\end{aligned}
$$

所以可以说,碰撞过程的本质是传递动量的过程.

### 1.3. 碰撞的分类

#### 1.3.1. 弹性碰撞与非弹性碰撞

（1）如果最终形变可以恢复，其他形式能（如弹性势能）又全部转化为系统的动能，这样系统的总机械能守恒，对应的是弹性碰撞的情况；

（2）如果形变没有完全恢复，则有一部分初始的动能损失掉了，这样的碰撞对应的就是非弹性碰撞；

（3）如果碰后形变完全不恢复，两个物体粘在一起，即两物体共速，这种情况下初始的动能损失最大，对应的就是完全非弹性碰撞．

#### 1.3.2. 对心碰撞与非对心碰撞
根据碰撞前后动量是否共线可以将碰撞分为两类 ．如果两个小球相碰，碰撞之前球的运动速度与两球心的连线在同一条直线上，碰撞之后两球的速度仍会沿着这条直线。这种碰撞称为正碰，也叫作对心碰撞或一维碰撞。如果碰撞前后两物体的速度不在一条直线上，这样的碰撞称为斜碰，也叫非对心碰撞。

### 1.4. 碰撞的合理性判断
（1）系统的动量守恒，即：$p_1+p_2=p^{\prime}{ }_1+p^{\prime}{ }_2$
（2）系统的机械能（碰撞前后势能一般不变，因此主要是动能）不会增加，即：

$$
E_{\mathrm{k} 1}+E_{\mathrm{k} 2} \geqslant E_{\mathrm{k} 1}^{\prime}+E_{\mathrm{k} 2}^{\prime}
$$

（3）碰撞物体的速度要符合实际情景，例如：
（1）碰前两物体同向运动，则后面物体的速度一定大于前面物体的速度，即 $v_{\text {后 }}>v_{\text {前，否则无法实 }}$现碰撞．
（2）同向运动的两物体碰撞后，原来在前面的物体的速度一定增大；若碰后两物体仍同向运动，则前面物体的速度大于或等于后面物体的速度，即 $v_{\text {前 }}^{\prime} \geqslant v_{\text {后，否则碰撞没有结束．}}$
（3）相向运动的两物体碰撞后，运动方向不可能都不改变，除非两物体碰撞后速度均为零．

## 3. 弹性碰撞
动能守恒且动量守恒.
### 3.1. 碰后速度的推导
![[Pasted image 20260203014815.png|300]]
动量守恒：$m_1 v_1+m_2 v_2=m_1 v_1^{\prime}+m_1 v_2^{\prime}$ 。
机械能守恒：$\frac{1}{2} m_1 v_1^2+\frac{1}{2} m_2 v_2^2=\frac{1}{2} m_1 v_1^{\prime 2}+\frac{1}{2} m_2 v_2^{\prime 2}$ 。
碰撞后速度：$v_1^{\prime}=\frac{\left(m_1-m_2\right) v_1+2 m_2 v_2}{m_1+m_2}, \quad v_2^{\prime}=\frac{\left(m_2-m_1\right) v_2+2 m_1 v_1}{m_1+m_2}$
### 3.2. 动碰静特例
![[Pasted image 20260203015245.png|300]]
动量守恒：$m_1 v_1=m_1 v_1^{\prime}+m_2 v_2^{\prime}$
能量守恒：$\frac{1}{2} m_1 v_1^2=\frac{1}{2} m_1 v_1^{\prime 2}+\frac{1}{2} m_2 v_2^{\prime 2}$
解得：$v_1^{\prime}=\frac{m_1-m_2}{m_1+m_2} v_1, \quad v_2^{\prime}=\frac{2 m_1}{m_1+m_2} v_1$

## 4. 完全非弹性碰撞
动量守恒+机械能损失最大.即形变完全不恢复.
### 特点：碰后共速
### 规律：

$$
\begin{cases}\text { 动量守恒: } & m_1 v_1+m_2 v_2=\left(m_1+m_2\right) v_{\text {共 }} \\ \text { 能量守恒: } & \frac{1}{2} m_1 v_1^2+\frac{1}{2} m_2 v_2^2=\frac{1}{2}\left(m_1+m_2\right) v_{\text {共 }}^2+\Delta E_{\text {损 }}\end{cases}
$$

解得：$\Delta E_{\text {损 }}=\frac{1}{2} \frac{m_1 m_2}{m_1+m_2}\left(v_1-v_2\right)^2$
### 完全非与弹性碰撞的联系
弹性碰撞的过程是完全非弹性的两倍.
$V_共=\frac{m_1 V_0}{m_1+m_2}$ (1)
对1球:
$$
\begin{aligned}
& \left.\begin{array}{l}
\text { I过程: } I_{12}=m_2 V_{\text {共 }}-0 \\
\text { II过程: } I_{12}=m_2 V_2-m_2 V_{\text {共 }}
\end{array}\right) \Rightarrow m_2 V_{\text {共 }}=m_2 V_2-m_2 V_{\text {共 (2) }} \\
& \text { (1)(2) ⇒ } V_2=2 V_共=\frac{2 m_1}{m_1+m_2} V_0
\end{aligned}
$$
对2球:
$\left.\begin{array}{l}I \text { 过程：} I_{21}=m_1 V_共-m_1 V_0 \\ I I \text { 过程：} I_{21}=m_1 V_1-m_1 V_{\text {共 }}\end{array}\right) \Rightarrow m_1 V_{\text {共 }}-m_1 V_0=m_1 V_1-m_1 V_{\text {共 }}$(3)

$$
\text { (1)(3) } \Rightarrow v_1=2 v_共-v_0=\frac{m_1-m_2}{m_1+m_2} v_0
$$
